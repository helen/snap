WARNING: This takes over your homepage and archives.== Description ==Snap is a plugin that turns your site into an easy-to-use photo sharing site. All you need to is log in, drag some images to upload, and they will appear on the homepage and in archives. <strong>Note that posts will not show alongside</strong>.* Images will show as being uploaded on the date the image was taken, if the data is available.<strong>Contribute to Snap</strong>This plugin is in active development <a href="https://github.com/helenhousandi/snap" target="_new">on GitHub</a>. If you'd like to contribute, pull requests are welcome!== Installation ==Manual:1. Upload the `snap` folder to your site's `/wp-content/plugins/` directory2. Activate the plugin through the 'Plugins' menu in WordPressFrom WordPress:1. Navigate to the Plugins > Add New screen.2. Search for 'Snap'3. Click Install and Activate Plugin when installation is complete.== Other Notes ==The following themes are photo-friendly and have been tested with Snap:* All default themes, such as Twenty Thirteen.* [Quiescent](http://devpress.com/themes/quiescent/) (Note: throws some notices with `WP_DEBUG` on.)== Changelog === 0.1 =* A first run of basic functionality.
